Увлажняющий мусс

Натуральный увлажняющий мусс для кожи лица. Содержит органические компоненты и экстракты растений для увлажнения и питания кожи. Обеспечивает длительное увлажнение, помогает справиться с сухостью и шелушением. Легкая текстура обеспечивает быстрое впитывание без ощущения липкости. Идеально подходит для ежедневного использования утром и вечером.

Состав:
- Органические экстракты алоэ вера и огурца
- Гиалуроновая кислота
- Натуральные масла жожоба и авокадо
- Витамин Е

Страна производства: Германия
Объем: 50 мл
Дополнительно:
- Не содержит парабенов и силиконов
- Не тестировался на животных
- Рекомендовано для всех типов кожи
- Для ежедневного использования
- Подходит для мужчин и женщин
